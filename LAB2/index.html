Course Catalog Description:

This course will cover the creation of Web pages and sites using HTML,CSS,Javascript and graphical applications. Networked multimedia distribution technologies are also explored. The design of effecive Web site including page layout,user interface design, graphic design,content flow and site structures as well as managment of Web site resources including intranet managment and design considerations and addressed. Students design and create a major Web site with multiple pages and cross-linked structures. Prerequesites: [none] Credit: 2-2-3 Semester Hours.

Course Outcomes

Students completing this course will be able to:

  • Recogize HTML,CSS and Javascript mark-up and code in a web page/application.
  • Select the proper mark-up tags or code to achieve a particular result
  • Identify improperly used mark-up and code
  • Produce modern standards compliant web pages
  • Deploy web pages to a public server
